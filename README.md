# Spring Boot Authentication System

A simple **login and registration system** built using **Spring Boot**, **MongoDB**, and **vanilla HTML/CSS/JavaScript**.

## Features
- User registration
- User login
- Password hashing (BCrypt)
- DTO-based API design
- Global exception handling
- Validation using Bean Validation
- Spring Security configuration
- Frontend integration using Fetch API

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- MongoDB
- HTML, CSS, JavaScript

## API Endpoints
- POST /register
- POST /login

## How to Run
1. Clone the repository
2. Configure MongoDB in `application.properties`
3. Run the Spring Boot application
4. Open frontend files in browser
